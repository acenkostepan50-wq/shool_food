# shool_food
1. Проект «SchoolFood» — это веб-приложение для учёта и контроля питания в школьной столовой.
Система предназначена для автоматизации процессов питания учащихся, учёта выданных блюд, контроля остатков продуктов и формирования отчётности.

В приложении реализованы три уровня доступа:
	•	ученик — просмотр меню, оплата питания, отметка о получении пищи, указание пищевых аллергий и оставление отзывов;
	•	повар — учёт выданных завтраков и обедов, контроль остатков продуктов, создание заявок на закупку;
	•	администратор — просмотр статистики, согласование заявок на закупку и формирование отчётов.
Проект реализован с использованием клиент-серверной архитектуры и предназначен для повышения прозрачности и эффективности работы школьной столовой.
2.  Инструкция по установке и запуску
1. Требования
	•	Node.js версии 18 или выше
	•	PostgreSQL 17
	•	Git
	•	Любой современный браузер
2. Установка проекта (команды прописываются в терминале)
Склонировать репозиторий:
git clone <https://github.com/leshiy877/shool_food.git>
Перейти в папку проекта:
cd schoolfood
3. Установка и запуск backend
Перейти в папку backend:
cd backend
Установить зависимости:
npm install

Запустить сервер:
node index.js
Backend будет доступен по адресу:
http://localhost:3001 (ссылка может немного отличаться, точнее будет написано в терминале)
4. Установка и запуск frontend
Открыть новый терминал и перейти в папку frontend:
cd frontend
Установить зависимости:
npm install
Запустить приложение:
npm start
Frontend будет доступен по адресу:
http://localhost:3000
5. Работа с системой
Для работы в системе используются тестовые учетные записи:
	•	ученик — логин 1, пароль 0000
	•	повар — логин 2, пароль 0000
	•	администратор — логин 3, пароль 0000
